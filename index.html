CI Project 4 - Full Stack Generative AI - Cre8AI.art

"Cre8AI.art" is a web application that offers users the opportunity to generate custom artwork utilizing the powerful DALLE-2 API by OpenAI. Once the AI-generated image is crafted, it's automatically presented as a public post, granting the user various options like viewing, editing its description, toggling its privacy settings, downloading, or even deleting the post. To maintain resource efficiency, user accounts are restricted to five image generations daily. Additionally, users can upload AI-generated images from other platforms. The website also encompasses an 'About' section, which admins can easily manage and modify, adding new pages as necessary through the admin panel. The primary target audience for "Cre8AI.art" comprises individuals with a keen interest in art and AI technology.

CI Project 3 - Python Terminal App - Quiz:

"KC-7's Quiz" is a Python-based command-line quiz application designed to help users familiarize themselves with a broad spectrum of functions. Upon initiating the quiz, users are prompted to enter their names, followed by a series of questions. Each input is meticulously validated for format accuracy. Connect 4 JS Game - FourStar.Pizza:

"FourStar.Pizza", an interactive web-based Connect 4 game where the primary objective is to earn four stars by aligning four pizzas on a 6x7 game board. Players take alternate turns placing their chosen pizza on the board, aiming to consecutively align four pizzas either horizontally, vertically, or diagonally. The game boasts a responsive design, ensuring compatibility across various devices. The underlying JavaScript code utilizes the DOMContentLoaded event to initialize the game and several functions to check for winning combinations, display available moves, and manage the game's logic. The game's intuitive design and engaging mechanics offer players a delightful and competitive experience.
